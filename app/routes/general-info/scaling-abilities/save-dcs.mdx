# Save DC Formulas (XPHB)

This reference lists all XPHB abilities that require save DCs. The standard formula is: **DC = 8 + Ability Modifier + Proficiency Bonus**

## Species/Races

### Aasimar

- **Necrotic Shroud**: DC = 8 + **CHA** + PB (Frightened condition)

### Dragonborn

- **Breath Weapon**: DC = 8 + **CON** + PB (Dexterity save for half damage)

## Feats

### Actor

- **Mimicry**: DC = 8 + **CHA** + PB (Wisdom Insight check to detect fake)

### Boon of Energy Resistance

- **Redirect Damage**: DC = 8 + **CON** + PB (Dexterity save or take 2d12 + CON)

### Poisoner

- **Poison Doses**: DC = 8 + **ability** + PB (Constitution save vs Poisoned + 2d8 damage)

### Shield Master

- **Shield Bash**: DC = 8 + **STR** + PB (Strength save vs push/prone)

### Telekinetic

- **Telekinetic Shove**: DC = 8 + **ability** + PB (Strength save vs forced movement)

## Class Features

### Barbarian - Subclasses

#### Berserker

- **Intimidating Presence** (14): DC = 8 + **STR** + PB (Wisdom save vs Frightened)

#### Path of the World Tree

- **Branches of the Tree** (6): DC = 8 + **STR** + PB (Strength save vs teleport)

### Fighter - Subclasses

#### Battle Master

- **Combat Superiority** (3): DC = 8 + **STR/DEX** + PB (various maneuvers)

#### Psi Warrior

- **Telekinetic Thrust** (7): DC = 8 + **INT** + PB (Strength save vs Prone/push)

### Monk

- **Monk's Focus** (2): DC = 8 + **WIS** + PB (various ki features)
- **Deflect Attacks** (3): DC = 8 + **WIS** + PB (Dexterity save for redirected damage)

### Paladin

- **Abjure Foes** (9): DC = 8 + **CHA** + PB (Wisdom save vs Frightened)

### Rogue

- **Cunning Strike** (5): DC = 8 + **DEX** + PB (various effects)

#### Assassin

- **Death Strike** (17): DC = 8 + **DEX** + PB (Constitution save vs doubled damage)

#### Soulknife

- **Rend Mind** (17): DC = 8 + **DEX** + PB (Wisdom save vs Stunned)

## Save DC Patterns

### Common Formula

**DC = 8 + Ability Modifier + Proficiency Bonus**

### By Ability Score

#### Strength (STR)

- Barbarian features (Berserker, World Tree)
- Shield Master

#### Dexterity (DEX)

- Dragonborn Breath Weapon (target saves with DEX)
- Rogue features (Cunning Strike, Death Strike, Rend Mind)

#### Constitution (CON)

- Boon of Energy Resistance
- Poisoner

#### Intelligence (INT)

- Psi Warrior features

#### Wisdom (WIS)

- Monk's Focus and features

#### Charisma (CHA)

- Aasimar Necrotic Shroud
- Actor feat
- Paladin Abjure Foes

### By Class

**Barbarians**: STR-based DCs
**Fighters**: STR/DEX (Battle Master), INT (Psi Warrior)
**Monks**: WIS-based DCs
**Paladins**: CHA-based DCs
**Rogues**: DEX-based DCs

### Notes

- Most save DCs scale with both the character's primary ability modifier and their proficiency bonus
- This ensures DCs remain challenging as characters level up
- Some abilities allow choice of ability modifier (e.g., Battle Master can use STR or DEX)
- Save types vary by effect: physical effects usually require STR/DEX/CON saves, while mental effects require WIS/CHA saves
